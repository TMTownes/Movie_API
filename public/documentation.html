The myFlix Documentation




Sign Up Login URL HTTP Method Login Response
Create New User (Register) /users POST A JSON object holding Model data about a user that was added
Login User /login POST A JSON object holding Model data about a user with hashed password authentication and bearer token for page authorization throughout the site
User Request User URL HTTP Method User Response
Update user info by Username /users/:Username PUT A JSON object holding Model data about a user that was updated
Add new Movie to FavoriteMovies List by MovieID (msg when complete) /users/Username/movies/:MovieID PUT A text message indicating the user's Favorite's List was updated.
DELETE movie from FavoriteMovies List /users/Username/movies/:MovieID DELETE A text message indicating a movie was removed from the user's Favorite's List
DELETE user profile (msg when complete) /users/:Username DELETE A text message indicating a user was removed.
Movie Request Movie URL HTTP Method Movie Response
Get a list of ALL movies /movies GET A JSON object holding Model data about all movies
Get a data on single movie /movies/:Title GET A JSON object holding Model data about a movie including ID, title, description, genre, director, imageURL, featured y/n
GET description of movie genre by name (i.e "Thriller") /movies/Genre/:Name GET A JSON object holding Model data about a specific genre
GET data about director by name /movies/Director/:Name GET A JSON object holding Model data about a director including bio, birth year, death year
