The myFlix Documentation



Movie Request Movie URL HTTP Method Movie Response
Get a list of ALL movies /movies GET A JSON object holding data about all movies
Get a data on single movie /movies/:title GET A JSON object holding data about a movie including ID, title, description, genre, director, imageURL, featured y/n
GET description of movie genre by name (i.e "Thriller") /movies/genre/:genre GET A JSON object holding data about a specific genre
GET data about director by name /movies/director/:name GET A JSON object holding data about a director including bio, birth year, death year
User Request User URL HTTP Method User Response
Create New User (Register) /users POST A JSON object holding data about a user that was added
Update user info by ID /users/:id PUT A JSON object holding data about a user that was updated
Add new Movie to Favorites List (msg when complete) /users/:id/:movieTitle PUT A text message indicating the user's Favorite's List was updated
DELETE movie from Favorites List /users/:id/:movieTitle DELETE A text message indicating a movie was removed from the user's Favorite's List
DELETE user profile (msg when complete) /users/:id DELETE A text message indicating a user was removed
